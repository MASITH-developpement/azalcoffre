1. Executive Summary

Vision

AZALPLUS est un ERP (logiciel de gestion d'entreprise) 100% gratuit destiné aux TPE françaises. Contrairement aux solutions existantes qui facturent des abonnements mensuels de 30€ à 200€, AZALPLUS offre gratuitement plus de 90 modules métier : facturation, comptabilité, CRM, gestion de stock, projets, RH, interventions, et bien plus encore.

Le modèle économique repose sur les flux financiers qui transitent par la plateforme : commissions sur les paiements Open Banking, vente de données anonymisées B2B, et services bancaires optionnels.

Historique et genèse

MASITH Développement est née d'un constat simple : les TPE françaises sont mal équipées en outils de gestion. Les solutions existantes sont soit trop complexes, soit trop chères pour des structures de 1 à 10 salariés.

Le fondateur, fort de son expérience dans le développement logiciel et la gestion d'entreprise, a conçu AZALPLUS comme une réponse à ce problème. L'idée centrale : rendre l'ERP gratuit en se rémunérant sur les flux financiers, créant ainsi un alignement d'intérêts entre l'éditeur et ses utilisateurs.

Mission et valeurs

Mission

Démocratiser l'accès aux outils de gestion professionnels pour toutes les TPE françaises, en supprimant la barrière du coût.

Valeurs fondamentales :

  • Gratuité réelle : Pas de freemium déguisé, pas de limites artificielles
  • Transparence : Modèle économique clair et compréhensible
  • Respect des données : Jamais de vente de données personnelles
  • Indépendance : Pas de dépendance à un partenaire unique

3. Le Produit : AZALPLUS

Description générale

AZALPLUS est un ERP (Enterprise Resource Planning) cloud conçu spécifiquement pour les TPE. Il regroupe tous les outils de gestion dont une petite entreprise a besoin, accessibles depuis n'importe quel navigateur web.

Plus de 90 modules métier

AZALPLUS propose plus de 90 modules fonctionnels, couvrant l'intégralité des besoins d'une TPE. Voici les principales catégories :

Comparaison de coût

Une TPE qui souhaiterait assembler une solution équivalente avec des outils payants dépenserait entre 200€ et 500€ par mois. AZALPLUS offre tout cela gratuitement.

Architecture technique

Stack technologique

  • Backend : Python 3.11, FastAPI, PostgreSQL 16
  • Frontend : Généré dynamiquement (No-Code engine)
  • Infrastructure : Cloud français, RGPD compliant
  • Sécurité : Chiffrement AES-256, authentification 2FA

Avantage compétitif : Architecture No-Code

AZALPLUS repose sur une architecture No-Code propriétaire. Chaque module métier est défini par un simple fichier de configuration (YAML), automatiquement transformé en :

  • Tables de base de données
  • API REST complète
  • Interface utilisateur
  • Workflows métier

Cette architecture permet de créer de nouveaux modules en quelques heures au lieu de plusieurs semaines, réduisant drastiquement les coûts de développement et de maintenance.

5. Modèle Économique

Principe fondamental

L'ERP est gratuit. AZALPLUS se rémunère sur les flux financiers qui transitent par la plateforme.

Ce modèle crée un alignement d'intérêts : AZALPLUS gagne quand les TPE utilisent activement le système et encaissent leurs factures.

Sources de revenus

Source 1 : Paiements Open Banking (Principal)

Chaque facture émise via AZALPLUS inclut un bouton "Payer maintenant". Le client final clique, est redirigé vers sa banque, et valide un virement instantané.

Élément Détail
Commission partenaire (Powens/Bridge) ~1.0%
Marge AZALPLUS 0.3 - 0.5%
Total facturé au client final 1.3 - 1.5%

Avantages vs carte bancaire

  • Moins cher : 1.3% vs 1.5-2.9% (Stripe, PayPal)
  • Instantané : Fonds reçus immédiatement vs 2-7 jours
  • Pas de chargeback : Virement irrévocable

Source 2 : Données anonymisées (Secondaire)

L'ERP collecte des données sur l'activité des TPE. Ces données, agrégées et anonymisées, ont une valeur pour :

  • Banques (scoring crédit, modèles de risque)
  • Assureurs (tarification, sinistralité)
  • Cabinets de conseil (études sectorielles)
  • Collectivités (politique économique)
Produit Prix
Étude sectorielle ("Plombiers IDF") 5 000 - 15 000€
Baromètre mensuel TPE 2 000 - 5 000€/mois
API données agrégées 20 000 - 50 000€/an

Activation : À partir de 5 000 TPE actives (masse critique pour représentativité).

Source 3 : Compte bancaire intégré (Optionnel)

Les TPE peuvent optionnellement activer un compte bancaire professionnel intégré à l'ERP (via partenaire BaaS type Swan).

Source Revenu AZALPLUS
Frais de compte 3-5€/mois par compte
Interchange carte Variable
Tap to Pay (encaissement) 0.2-0.3% des transactions

Activation : Phase 2, après validation du modèle paiement. Estimation : 20-30% des TPE activeront cette option.
