# Raccourcis Clavier

Utilisez ces raccourcis pour naviguer plus rapidement dans AZALPLUS.

## Navigation globale

| Raccourci | Action |
|-----------|--------|
| `Ctrl + K` | Ouvrir la recherche rapide |
| `Ctrl + H` | Acceder au tableau de bord |
| `Ctrl + N` | Nouveau (dans le module actif) |
| `Ctrl + S` | Enregistrer |
| `Echap` | Fermer la fenetre modale |

## Dans les listes

| Raccourci | Action |
|-----------|--------|
| `Fleche Haut/Bas` | Naviguer dans la liste |
| `Entree` | Ouvrir l'element selectionne |
| `Suppr` | Supprimer l'element selectionne |
| `Ctrl + A` | Selectionner tous les elements |
| `Ctrl + Click` | Selection multiple |
| `Shift + Click` | Selection continue |

## Dans les formulaires

| Raccourci | Action |
|-----------|--------|
| `Tab` | Champ suivant |
| `Shift + Tab` | Champ precedent |
| `Ctrl + Entree` | Enregistrer et fermer |
| `Ctrl + D` | Dupliquer l'enregistrement |

## Edition de texte

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Z` | Annuler |
| `Ctrl + Y` | Refaire |
| `Ctrl + C` | Copier |
| `Ctrl + V` | Coller |
| `Ctrl + X` | Couper |

## Modules rapides

| Raccourci | Module |
|-----------|--------|
| `Alt + C` | Clients |
| `Alt + D` | Devis |
| `Alt + F` | Factures |
| `Alt + I` | Interventions |
| `Alt + P` | Produits |

## Actions documents

| Raccourci | Action |
|-----------|--------|
| `Ctrl + P` | Imprimer / Generer PDF |
| `Ctrl + E` | Envoyer par email |
| `Ctrl + M` | Ajouter une note |

## Calendrier

| Raccourci | Action |
|-----------|--------|
| `T` | Aller a aujourd'hui |
| `Fleche Gauche` | Periode precedente |
| `Fleche Droite` | Periode suivante |
| `J` | Vue jour |
| `S` | Vue semaine |
| `M` | Vue mois |

## Aide

| Raccourci | Action |
|-----------|--------|
| `F1` | Ouvrir l'aide contextuelle |
| `?` | Afficher les raccourcis |
| `Ctrl + /` | Rechercher dans l'aide |

## Personnalisation

Vous pouvez personnaliser certains raccourcis dans :
**Parametres > Preferences > Raccourcis clavier**

## Conseils

- Les raccourcis avec `Ctrl` correspondent a `Cmd` sur Mac
- Certains raccourcis peuvent etre desactives selon vos permissions
- Les raccourcis fonctionnent uniquement en dehors des champs de saisie
