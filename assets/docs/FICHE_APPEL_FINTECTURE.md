## 2. SOLUTIONS FINTECTURE

### 2.1 Virement Immédiat (principal)
- Paiement par initiation de paiement (Open Banking)
- Temps réel : 2-3 secondes (max 10s)
- Irrévocable = **0 chargeback**
- Authentification forte (SCA) native
- Disponible 24/7

### 2.2 Virement Intelligent
- Virement bancaire classique
- Réconciliation automatique
- Pour les paiements B2B

### 2.3 BNPL B2B
- Paiement différé pour entreprises
- Couplé assurance impayés (Allianz Trade)

## 7. NOTRE PROPOSITION AZALPLUS

### 7.1 Présentation AZALPLUS
- **ERP 100% gratuit** pour TPE françaises
- **90+ modules** : Facturation, Compta, CRM, Stock, Projets...
- **Cible** : Artisans, Commerçants, Consultants, Prof. libérales
- **Modèle** : Gratuit si paiement intégré, sinon 1.5% CA

### 7.2 Volume potentiel
- **Cible** : 10 000 TPE à 3 ans
- **CA moyen TPE** : 80 000€/an
- **Volume paiements** : 800M€/an potentiel

### 7.3 Ce qu'on cherche
1. **API paiement par virement** intégrée dans nos factures
2. **Bouton "Payer maintenant"** sur devis/factures
3. **Réconciliation automatique** facture ↔ paiement
4. **Commission partenaire** sur les transactions
5. **Support technique** pour l'intégration

### 7.4 Avantages pour Fintecture
- **Acquisition client** via notre base TPE
- **Volume récurrent** (factures mensuelles)
- **Secteur TPE** peu digitalisé = fort potentiel
- **Intégration native** = meilleure adoption

## 8. POINTS DE NÉGOCIATION

### 8.1 Notre force
- Distribution gratuite vers les TPE
- Intégration native (pas de plugin tiers)
- Volume potentiel important (800M€/an)
- Secteur sous-pénétré par les fintechs

### 8.2 Ce qu'on veut négocier
| Point | Objectif | Acceptable |
|-------|----------|------------|
| Commission partenaire | 30% de leur marge | 20% minimum |
| Frais mensuels | 0€ | Max 100€/mois |
| Frais intégration | 0€ | Accompagnement gratuit |
| Support | Dédié | Prioritaire |
| Sandbox | Illimité | 3 mois minimum |

### 8.3 Red flags à surveiller
- Exclusivité demandée (refuser)
- Engagement volume minimum élevé
- Frais cachés (setup, maintenance)
- Délais d'intégration > 3 mois
