# Questions Frequentes (FAQ)

## Compte et Connexion

### Comment reinitialiser mon mot de passe ?
1. Sur la page de connexion, cliquez sur **Mot de passe oublie**
2. Saisissez votre adresse email
3. Consultez votre boite mail
4. Cliquez sur le lien de reinitialisation
5. Choisissez un nouveau mot de passe

### Puis-je avoir plusieurs utilisateurs ?
Oui, AZALPLUS est multi-utilisateurs. Contactez votre administrateur pour creer de nouveaux comptes.

### Comment modifier mes informations personnelles ?
Cliquez sur votre nom en bas du menu lateral, puis **Mon profil**.

---

## Devis et Factures

### Comment creer un devis ?
1. Menu lateral > **Devis**
2. Cliquez sur **+ Nouveau**
3. Selectionnez un client
4. Ajoutez vos lignes de produits
5. Enregistrez

### Comment convertir un devis en facture ?
1. Ouvrez le devis accepte
2. Cliquez sur **Convertir en facture**
3. Verifiez les informations
4. Validez

### Puis-je modifier une facture envoyee ?
Non, une facture envoyee ne peut pas etre modifiee. Vous devez :
- Creer un avoir
- Emettre une nouvelle facture

### Comment appliquer un code promo ?
1. Dans le devis ou facture
2. Saisissez le code dans le champ **Code promo**
3. Cliquez sur **Appliquer**
4. La remise est calculee automatiquement

### Comment gerer la TVA multi-taux ?
Chaque ligne de produit peut avoir un taux de TVA different :
1. Lors de la creation du produit, definissez son taux
2. Ce taux est applique automatiquement
3. La ventilation TVA est calculee par taux

## Clients

### Comment importer mes clients ?
1. **Parametres > Import**
2. Telechargez le modele CSV
3. Remplissez vos donnees
4. Importez le fichier

### Comment suivre les impayes ?
1. Tableau de bord > **Factures en retard**
2. Ou : Clients > Balance agee

### Comment fusionner deux fiches clients ?
1. Ouvrez le client a conserver
2. **Actions > Fusionner**
3. Selectionnez le doublon
4. Validez la fusion

---

## Paiements

### Comment enregistrer un paiement ?
1. Ouvrez la facture concernee
2. Cliquez sur **Enregistrer paiement**
3. Saisissez le montant et la date
4. Validez

### Comment gerer un paiement partiel ?
Saisissez le montant recu. La facture passe en **PARTIALLY_PAID** et le reste a payer est mis a jour.

### Comment annuler un paiement ?
1. Ouvrez la facture
2. Section **Paiements**
3. Cliquez sur **Annuler** a cote du paiement
4. Confirmez l'annulation

---

## Interventions

### Comment planifier une intervention ?
1. Menu > **Interventions**
2. Cliquez sur **+ Nouveau**
3. Selectionnez le client et le type
4. Definissez la date et l'horaire
5. Assignez un technicien

### Comment suivre une intervention en cours ?
Le calendrier affiche le statut en temps reel :
- Bleu : Planifiee
- Orange : En cours
- Vert : Terminee

## Documents

### Quels formats de fichiers sont acceptes ?
- Documents : PDF, Word, Excel
- Images : JPG, PNG, GIF
- Archives : ZIP

### Quelle est la taille maximale ?
10 Mo par fichier.

### Comment attacher un document ?
1. Ouvrez l'enregistrement
2. Section **Documents**
3. Glissez-deposez ou cliquez sur **Parcourir**

## Export et Rapports

### Comment exporter mes donnees ?
1. Dans n'importe quelle liste
2. Appliquez vos filtres
3. Cliquez sur **Exporter**
4. Choisissez CSV ou Excel

### Comment generer un rapport ?
1. Menu > **Rapports**
2. Selectionnez le type de rapport
3. Definissez la periode
4. Generez

## Performance et Problemes

### L'application est lente, que faire ?
1. Verifiez votre connexion internet
2. Videz le cache du navigateur
3. Fermez les onglets inutilises
4. Contactez le support si le probleme persiste

### Je ne vois pas tous les modules
Vos permissions peuvent etre limitees. Contactez votre administrateur.

### Une erreur s'affiche, que faire ?
1. Notez le message d'erreur
2. Rafraichissez la page
3. Si le probleme persiste, contactez le support avec :
   - Le message d'erreur
   - Les etapes pour reproduire le probleme
   - Votre navigateur et version
